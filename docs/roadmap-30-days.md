# 30-Day Delivery Roadmap

## Week 1 — Foundation
- Initialize Laravel + Inertia (React) + Tailwind
- Configure PostgreSQL, Redis, and queue worker
- Set up Sanctum/Fortify auth flows (login/register/reset)
- Implement roles + permission gates/policies

## Week 2 — Core Content
- Build Projects CRUD + filters + publish workflow
- Build Blog CMS (categories/tags, slugging, draft/publish)
- Add media upload handling for featured images
- Add public pages for projects/blog listings and details

## Week 3 — Dashboard + Analytics
- Implement visit tracking middleware/table writes
- Build aggregation queries for KPIs and trends
- Create admin dashboard widgets/charts with Recharts
- Add contact inbox panel and status transitions

## Week 4 — Hardening + Launch
- Add SEO metadata, sitemap, robots, JSON-LD
- Add reCAPTCHA + endpoint throttling for contact
- Add activity/audit logs for critical actions
- Smoke test, seed demo data, and deploy to production target
